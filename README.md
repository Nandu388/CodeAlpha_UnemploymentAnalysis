# Unemployment Analysis with Python

## Project Overview

This project analyzes unemployment trends in India using Python. The analysis focuses on unemployment rates across different regions and explores the impact of Covid-19 on employment patterns.

## Objectives

* Analyze unemployment rate data
* Perform data cleaning and preprocessing
* Visualize unemployment trends
* Study the impact of Covid-19 on unemployment
* Identify regional and economic patterns

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset Information

The dataset contains:

* Region
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area

## Data Analysis Performed

### Data Cleaning

* Checked missing values
* Inspected data types
* Generated statistical summaries

### Data Visualization

* Regional unemployment analysis
* Correlation heatmap
* State-wise unemployment distribution
